# Cadastro de conectores

1 - Todos os conectores devem possuir o parametro "nome.cliente" com seu respectivo nome do cliente. Ex: "nome.cliente": "vale"
2 - Adicone o arquivo de credenciais do cliente na pasta: /srv/main/resources/credentials/vale.json


# Executando
`mvn clean install`
`mvn spring-boot:run`
